Download e Installazione

macOS App (Consigliato)

App nativa con Java runtime integrato

61 MB macOS 10.15+

JAR Eseguibile

Versione cross-platform

~2 MB Java 17+

Installazione macOS

  1. Scarica BachecaAnnunci-1.0.dmg
  2. Apri il file DMG
  3. Trascina l'app nella cartella Applications
  4. Esegui dal Launchpad

Esecuzione JAR

  1. Installa Java 17+ se necessario
  2. Scarica BachecaAnnunci.jar
  3. Esegui: java -jar BachecaAnnunci.jar
